# Aprender.IO - Objeto Virtual de Aprendizagem (OVA)

## 📌 Descrição
O **Aprender.IO** é um Objeto Virtual de Aprendizagem (OVA) desenvolvido como parte de um projeto de 
**Iniciação Científica (IC)** no **IFG - Campus Inhumas**. 
A plataforma foi concebida para ser um ecossistema educacional digital, 
focado em auxiliar o processo de ensino e aprendizagem de diversas disciplinas da grade curricular do instituto.

## 🎯 Objetivo
O projeto visa combater dificuldades de aprendizagem através de uma interface interativa e acessível. 
* **Problema:** Disciplinas com alto índice de complexidade demandam ferramentas complementares ao ensino em sala de aula.
* **Solução:** Centralizar conteúdos modulares que permitam ao aluno estudar de forma autônoma e dinâmica.
* **Público-alvo:** Alunos do IFG Campus Inhumas.

## 🚀 Tecnologias Utilizadas
O projeto utiliza tecnologias web fundamentais para garantir leveza e compatibilidade:
- ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) - Estruturação de conteúdo.
- ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) - Estilização e design responsivo.
- ![JavaScript](https://img.shields.io/badge/javascript-%23F7DF1E.svg?style=for-the-badge&logo=javascript&logoColor=black) - Lógica da interface e interatividade.

## 📂 Estrutura do Projeto
A organização atual do repositório reflete a separação entre a estrutura de hospedagem e o núcleo da aplicação:

```text
├── aprenderio/         # Diretório principal da aplicação
│   ├── css/            # Estilizações específicas
│   ├── js/             # Scripts e lógica de funcionamento
│   └── assets/         # Imagens e recursos visuais
├── index.html          # Ponto de entrada principal da interface
└── README.md           # Documentação do projeto

```

## ⚙️ Funcionalidades

* **Navegação Modular:** Página inicial estruturada para receber diferentes disciplinas.
* **Módulo de Lógica de Programação:** (Em desenvolvimento) Conteúdo focado no ensino de algoritmos e fundamentos de programação.
* **Interface Responsiva:** Adaptada para diferentes dispositivos (computadores e dispositivos móveis).
* **Hospedagem de Interface:** Estrutura otimizada para carregamento rápido dos componentes educacionais.

## ▶️ Como Executar o Projeto

Para visualizar o projeto localmente:

1. **Clone o repositório:**
```bash
git clone [https://github.com/JaoVPimenta/Projeto-IC.git](https://github.com/JaoVPimenta/Projeto-IC.git)

```


2. **Navegue até a pasta:**
```bash
cd Projeto-IC

```


3. **Abra o arquivo principal:**
Basta abrir o arquivo `index.html` em qualquer navegador moderno.

## 💡 Exemplos de Uso

Ao acessar a plataforma, o aluno encontrará o menu principal com os módulos disponíveis. Atualmente, ao selecionar o módulo de **Lógica de Programação**, o usuário terá acesso a trilhas de conhecimento e exercícios práticos voltados para o desenvolvimento do pensamento algorítmico.
